# FUT12 Revival

FUT12 Revival is an open-source reverse engineering project whose goal is to recreate the backend services required to make FIFA 12 Ultimate Team playable again.

The project aims to:

- emulate EA Blaze services
- recreate Ultimate Team backend logic
- support RPCS3
- support original PS3 hardware where technically possible
- document the FIFA 12 protocol

This repository contains no EA assets or copyrighted game files.
